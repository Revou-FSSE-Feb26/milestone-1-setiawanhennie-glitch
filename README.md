# Hennie's Portfolio

> A personal portfolio website for an aspiring software engineer transitioning from biotechnology into tech.

---

## 📖 Overview of the Website

The website showcases my portfolio with the following sections:

- **Home**: This section contains a brief bio including my profile photo.
- **About Me**: It tells my background in education, work experience I have gained so far, current focus, technical & non-technical skills plus my interests.
- **Project Highlights**: I listed featured projects including a World Happiness Analysis dashboard built with Streamlit and the ongoing Nusa Match app for Indonesian SMEs.
- **Contact Section**: My social links (Email, LinkedIn, and GitHub) and an interactive contact form for collaboration inquiries or questions.

The site is designed to be lightweight, easy to navigate, and serves as a foundation for future enhancements as my web development skills grow.

---

## ✨ Features Implemented

### Navigation & Structure
- Fixed top-right navigation bar with smooth anchor links to sections: **About Me**, **Projects**, **Contact**
- Semantic HTML structure with `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` tags
- Horizontal rule (`<hr>`) dividers for clear visual separation between sections

### Content Sections
- **Home**: Introduction with profile image, headline, and bio
- **About Me**: 
  - Personal story and motivation
  - Education, work experience, and current learning focus
  - Categorized skills: Technical, Non-Technical
  - Personal Interests
- **Projects**: 
  - Project cards with title, description, and external links
  - Placeholder for upcoming projects 
- **Contact**: 
  - E-mail and social media with embedded links 
  - HTML form with validation: First Name, Last Name, Email, Birthday, and Message

### UI & Styling
- Inline CSS for quick prototyping:
  - Fixed navigation with dark background and rounded corner
  - Flexbox layout for navigation menu and contact links
  - Table-based layout for home section (profile image + text)
- Responsive viewport meta tag for mobile compatibility
- Emoji-enhanced headings for visual engagement

### Interactivity
- Clickable anchor links for in-page navigation
- Functional contact form with `required` field validation
- External links to GitHub, LinkedIn, and project demos
